# Breast Cancer Prediction using Decision Tree

## Project Overview
This project predicts whether a breast tumor is **Benign** or **Malignant** using a Decision Tree Classifier.

## Algorithm
Decision Tree Classifier

## Technologies
- Python
- Scikit-learn
- Streamlit
- NumPy
- Joblib

## Model Performance
- Accuracy: 94.74%
- Precision: 95.77%
- Recall: 95.77%
- F1 Score: 95.77%
